## Installieren

```bash
git clone https://github.com/Bylxly/netcontrol
```
```bash
cd netcontrol
```
```bash
chmod +x install_netcontrol.sh
```
```bash
sudo ./install_netcontrol.sh
```

## Deinstallieren

```bash
sudo ./uninstall_netcontrol.sh
```

## Befehle

```bash
netcontrol [OPTIONS]
```

```bash 
man netcontrol 
```
